## kintoneアプリの使い方
* template内の「連携ルックアップ.zip」をテンプレートとしてkintoneに展開。下記5アプリが展開されます。
  * 大分類マスタ
  * 中分類マスタ
  * 小分類マスタ
  * 商品マスタ
  * 売上登録
* data内の「大分類マスタ.csv」「中分類マスタ.csv」「小分類マスタ.csv」「商品マスタ.csv」をそれぞれのアプリに読み込ませると初期データを入れることができます。
